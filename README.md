## 🏆 Internship Project Submission

---

## 📌 Project Overview

The **Stock Portfolio Tracker Pro** is a Python-based desktop application that allows users to track and manage stock investments using a modern graphical user interface (GUI).

The application fetches live stock prices using Yahoo Finance API and provides real-time portfolio calculations, data visualization, and file export functionality.

This project demonstrates practical implementation of Python fundamentals along with GUI development and API integration.

---

## 🎯 Objectives

* Develop a user-friendly stock portfolio management system
* Integrate live stock price data
* Perform real-time investment calculations
* Visualize portfolio distribution
* Export portfolio data to CSV

---

## 🚀 Features

✔ Live stock price fetching using `yfinance`
✔ Add multiple stocks with quantity
✔ Auto calculation of total investment
✔ Delete selected stock option
✔ Pie chart visualization of portfolio distribution
✔ Save portfolio data to CSV file
✔ Modern dark theme GUI using Tkinter

---
